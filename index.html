Política de Privacidade

A presente Política de Privacidade descreve como a iPoolCare coleta, usa e protege as informações fornecidas pelos usuários ao utilizar nosso aplicativo.

1. Informações Coletadas

Nós coletamos apenas as seguintes informações pessoais para fins de acesso e uso do Aplicativo:

  • Nome: Utilizado para identificação do usuário dentro do sistema.
  • E-mail: Utilizado para o registro e comunicação com o usuário.
  • Senha: O sistema gera automaticamente uma senha inicial, que o usuário pode alterar posteriormente.

Esses dados são essenciais para criar e manter a conta de acesso do usuário ao Aplicativo.

2. Uso das Informações

As informações coletadas são utilizadas apenas para:

  • Criar e gerenciar a conta de usuário no Aplicativo.
  • Permitir o acesso seguro ao sistema por meio de login.
  • Enviar comunicações relacionadas ao uso do Aplicativo, como atualizações e informações relevantes.

3. Segurança dos Dados

Adotamos medidas de segurança apropriadas para proteger as informações dos usuários contra acessos não autorizados, alterações, divulgação ou destruição. As senhas são criptografadas para garantir a segurança e privacidade.

4. Alteração de Dados

Os usuários podem, a qualquer momento, alterar suas senhas diretamente no Aplicativo, garantindo um controle total sobre o acesso à conta.

5. Compartilhamento de Informações

As informações pessoais dos usuários não serão compartilhadas com terceiros, exceto quando for necessário para cumprir obrigações legais ou em conformidade com os regulamentos aplicáveis.

6. Alterações nesta Política

Reservamo-nos o direito de atualizar esta Política de Privacidade de tempos em tempos. Recomendamos que os usuários revisem periodicamente esta página para se manterem informados sobre qualquer alteração.

7. Contato

Caso tenha qualquer dúvida sobre esta Política de Privacidade ou sobre como tratamos suas informações, entre em contato conosco através do e-mail: privacidade@ipoolcare.com.
